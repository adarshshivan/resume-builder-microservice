A full-stack cloud project powered by `AWS Lambda`, `API Gateway`, `DynamoDB`, and a lightweight frontend hosted on `Vercel`.

## Project Overview

The Resume Builder is a fully functional resume creation platform that allows users to:

- Enter and preview resume details live
- Save resume data securely
- Update existing resumes
- Load resumes by email
- Generate downloadable PDF files

The application is designed using a serverless architecture, ensuring high scalability, minimal cost, and zero backend maintenance.

## Key Features

- Real-time resume preview
- Create, read, and update resumes
- PDF generation using AWS Lambda
- Pre-signed S3 URLs for secure downloads
- Automatic fallback into Demo Mode if AWS backend is offline
- Developer-friendly backend folder with all Lambda scripts
